## Courses

### CS 1301 - Introduction to Computing
**Semester:** Fall 2025  
**Focus:** Python programming fundamentals and robotics

A hands-on introduction to programming using Python and the iRobot Create® 3 educational robot. Course covers fundamental programming concepts including variables, control structures, functions, and object-oriented programming through real-world robotic applications.

**Topics Covered:**
- Event-driven programming
- Sensor data processing and integration
- Autonomous navigation algorithms
- Pathfinding and maze-solving techniques
- Object-oriented design
- Asynchronous programming

**Key Projects:**
- Password management system using robot inputs
- Autonomous object tracking and following
- Self-parking robot with gap detection
- Maze navigation using flood-fill algorithm
